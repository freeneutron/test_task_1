Мозаика
На главной странице меню из 2х пунктов:
- рисовать
- список сохраненных досок
Рисование:
Есть доска 20х20 клеток.
Пользователь, кликая на клетку доски, меняет ее состояние в такой последовательности: заполнена красным, заполнена желтым, заполнена зеленым, пустая.
Фиксируется время работы над “рисунком”.
Есть кнопка “сохранить” по нажатию на которую появляется предложение ввести имя рисунка, при вводе имемни и нажатии кнопки ОК “рисунок” сохраняется в БД.
Если “рисунок” с таким именем уже есть, то предлагается его перезаписать, либо ввести другое имя.
Список сохраненных досок:
Таблица:
- название
- дата создания
- дата последней модификации
- время работы над “рисунком”
- действия: загрузить, удалить
Требования к выполнению задания:
- ООП (можно использовать фреймворк)
- Логика на PHP
- Интерфейс на jquery
- Обмен данными используя подход ajax
Нужно будет приложение разместить:
1. Бесплатный хостинг / тестовый период
2. GitHub / BitBucket
